# User Resources

    POST login
    
## Description
Logs in a user.

***

## Payload

Essential information 

- **email** - User email address.
- **password** - Password associated with this account.

***

## On Success

Status Code `302`

Redirects To: [**[`GET`] user/profile**](endpoints/users/GET_user_profile.md)
    
***

## Errors

`500` - All known errors.
